Appel aux orateurs

L'appel aux orateurs est ouvert. La date de clôture est le dimanche 14 juillet 2014.

Précisions :

  • Les sessions sont principalement en français, ou in English
  • Tous les slots sont d'une heure, mais une session peut occuper plusieurs slots.
  • Le format de la session est libre (académique, atelier, interactif...) : surprenez votre auditoire !
  • L'orateur s'engage à remettre une version PDF de son support, afin que Soft-Shake.ch puisse les mettre gracieusement à disposition des participants.
  • L'orateur a son inscription à l'événement offerte. En cas de session avec plusieurs orateurs, seul l'orateur principal se voit offrir sa place.
  • Le budget particulièrement réduit de la conférence ne nous permet malheureusement pas de prendre en charge les frais des orateurs. Merci de votre compréhension.

Vous pouvez soumettre des sessions pour chacun des tracks identifiés :

  • Agilité (track-agile@soft-shake.ch)
  • Big Data / NoSQL (track-bigdata@soft-shake.ch)
  • IA / Robotique (track-robotique@soft-shake.ch)
  • Java (track-java@soft-shake.ch)
  • Microsoft (track-microsoft@soft-shake.ch)
  • Mobilité (track-mobile@soft-shake.ch)
  • Programmation Fonctionnelle (track-functional@soft-shake.ch)
  • Web [JS / CSS3 / HTML5] (track-web@soft-shake.ch)

Sans oublier le track Incubateur qui a pour vocation d'héberger toute session ne s'accordant pas avec les autres tracks. N'hésitez pas à soumettre vos sessions : du moment qu'il s'agit d'un sujet relatif au développement logiciel dans son ensemble (technologies, pratiques, méthodologies, rôles...), il n'y a pas de hors sujet !

Voici une liste d'exemples de sujets que nous souhaiterions voir proposés:

  • Réseaux sociaux, Community Management
  • Présentation Zen
  • Création d'entreprise, retour d'expérience sur la vie d'indépendant
  • La négociation
  • IA (Programmation par Contraintes, SMA, logique floue...)
  • Maitriser son IDE (Eclipse, IntelliJ, XCode...)
  • Robotique, domotique / automation, embarqué
  • Les imprimantes 3D
  • Open Source
  • UX, ergonomie, design...
  • DevOps
  • Core Protocols
  • Radical Management
  • PNL
  • Cloud
  • Management par les femmes
  • ToastMasters
  • Persistance (SGBDR, NoSQL, BigData...)
  • Gamification
  • Cryptographie, sécurité...
  • Présentation de langages : R, SmallTalk, etc...
  • Virtualisation (VMWare, Vagrant, ...)
  • Architecture du code, "Clean Code"..."
  • Hygiène de vie (professionnelle), ergonomie du poste de travail...
  • Apprendre à apprendre
  • ...

Nous espérons que cela achève de vous convaincre qu'il n'y a pas de mauvais sujet, surtout dans l'Incubateur... Alors soumettez vos sujets !

Si vous avez des questions relatives à un track en particulier, contactez directement les responsables du track via l'adresse email indiquée ci-dessus entre parenthèses et track-incubator@soft-shake.ch pour l'Incubateur.

Pour tout autre question, vous pouvez contacter l'équipe d'organisation : orga@soft-shake.ch

Nous espérons à bientôt!
