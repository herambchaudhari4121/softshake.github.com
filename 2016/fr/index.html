"Call For Paper"

Envie de parler d'un sujet, de faire un retour sur expérience,
N'hésite pas, soumets nous ton sujet.
N’hésitez pas à vous porter candidat si vous souhaitez partager un sujet :

Les incontournables :
java, agilité, Microsoft, mobilité, web / JavaScript, Programmation Fonctionnelle, NoSQL / BigData, Machine Learning / Intelligence Artificielle, IoT / Robotique

Les fondamentaux, ancestraux, novateurs, confidentiels, intrigants (track incubateur) :
Design UX, Assembleur, Management, Smalltalk, Oracle, Gamification, Lean Startup, bien utilisé mon IDE, tel outil, telle technologie…
Et tout ce qui touche de près ou de loin à notre métier !

Cette année, nous allons utiliser un nouveau service (kora.li)

Pour soumettre les sessions:

Je n'ai pas de compte

Enregistre-toi sur le site kora.li

Valide ton compte avec le mail d’activation que tu recevras de kora.li

J'ai déjà un compte

Connecte-toi sur http://kora.li/signin

Choisit “Conferences” sur le menu de gauche
Cherche “Softshake” et sélectionne la conférence
Clique sur le bouton “submit” en bas
Remplit le formulaire en sélectionnant “create new session” pour le champ “Session”,
Remplit et valide.

Ta session est maintenant enregistrée pour softshake :
- Nous (organisateurs de softshake) pouvons la consulter
- Tu peux la modifier à ta guise
- Tu peux l’utiliser pour la soumettre dans d’autres conférences…
